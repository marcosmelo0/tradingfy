## 🌟 O que é o TradingFy?

O **TradingFy** não é apenas um diário de trading; é um hub de inteligência institucional. Desenvolvido para traders que buscam a consistência em mesas proprietárias (prop firms), ele transforma dados brutos de histórico em insights comportamentais e métricas de risco avançadas.

### 🎯 Diferenciais
- **Foco em Prop Firms**: Métricas específicas para Drawdown, Payout e regras de consistência.
- **Design Institucional**: Interface em Dark Mode de alto contraste, otimizada para longas sessões de análise.
- **Ecossistema Completo**: De calendários econômicos a monitoramento de risco em tempo real.

## 🛠️ Stack Tecnológica

- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Estilização**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Gráficos**: [Recharts](https://recharts.org/)
- **Backend/DB**: [Supabase](https://supabase.com/) (PostgreSQL + Auth + Edge Functions)
- **Pagamentos**: [Stripe API](https://stripe.com/)
- **Ícones**: [Lucide React](https://lucide.dev/)
- **Datas**: [React Datepicker](https://reactdatepicker.com/) + [Date-fns](https://date-fns.org/)

## 📦 Como rodar o projeto

1. **Clone o repositório**
   ```bash
   git clone https://github.com/seu-usuario/tradingfy.git
   ```

2. **Instale as dependências**
   ```bash
   npm install
   ```

3. **Configure as Variáveis de Ambiente**
   Crie um arquivo `.env` na raiz com:
   ```env
   VITE_SUPABASE_URL=sua_url
   VITE_SUPABASE_ANON_KEY=sua_chave
   ```

4. **Inicie o servidor de desenvolvimento**
   ```bash
   npm run dev
   ```
